Digital Buddy is a full-stack productivity and personal management web application built with React + Vite and Firebase. It helps users organize tasks, receive smart reminders, track hydration, maintain a secure diary, and generate AI-based schedules in one unified platform.

It includes secure authentication, cloud data sync, sound customization, recycle bin recovery, and responsive modern UI for daily productivity workflows.
